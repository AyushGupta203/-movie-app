# 🎬 Movie Search App

A React-based movie browsing app using OMDB API with optimized performance and modern UI.

## 🚀 Features
- Search movies using OMDB API
- Movie detail page with routing
- Watchlist using Context + useReducer
- Optimized image loading (improved LCP from 3.1s → 1.3s)
- Responsive UI with Tailwind CSS

## 🛠 Tech Stack
- React.js
- React Router
- Context API + useReducer
- Tailwind CSS
- OMDB API

## ⚡ Performance Optimization
- Hybrid image loading (eager + lazy)
- Reduced Largest Contentful Paint (LCP)
- Improved initial rendering speed
